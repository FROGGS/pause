Admin stuff

Note: This page is an ancient relict of early incarnations of the PAUSE. Most of the scripts below redirect to the new interface that has a menu of its own. Forget this page, it isn't maintained anymore.

Publicly accessible directories

  • The homepage
  • List of people (big and slow)
  • ftp://pause.perl.org/incoming/

The user's directory (user password required)

  • /pub/PAUSE/ via http
    • 04pause.html
  • /incoming/ via http
  • add files to your home directory or
  • edit your name, email, and homepage address or
  • change your password
  • delete your files or
  • expedite an urgent mirror (or just inspect the mirrored items) or
  • see the last lines of the pause daemon logfile (the number in the path is the number of bytes from the file end) or
  • edit an archive

The admin directory (admin password required)

  • add a new user
  • as admin, edit an existing user
  • as admin, change a user's password
  • as admin, add an uri to a user's account
  • as admin, delete files
  • as admin, add an archive
  • Server status (with refresh), Perl Status
  • Server configuration
  • Server connectivity
